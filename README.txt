==============================
  DONEFY - AI-Powered Productivity
  Version 1.0
==============================

FIRST TIME SETUP
-----------------
1. Install Python 3.10+  →  https://python.org/downloads
   !! Check "Add to PATH" during install !!

2. Install Node.js LTS   →  https://nodejs.org

3. Double-click: setup.bat
   (installs everything and opens the app)


DAILY USE
----------
Double-click: run.bat


BUILD .EXE (to share with others)
-----------------------------------
Double-click: build-exe.bat
→ Creates: backend\dist\DONEFY\DONEFY.exe

Share the entire "DONEFY" folder (zip it).
The person receiving it does NOT need Python or Node.js.


FEATURES
---------
✓ Task management — add, edit, delete, complete
✓ Date-based organization (Today / Upcoming / No Date)
✓ Priority levels: High / Medium / Low
✓ Due dates and reminders
✓ Voice input (mic button)
✓ AI task capture — speak freely, AI structures it
✓ Photo attachments
✓ Monthly calendar view
✓ AI Assistant (chat, prioritize, suggest)
✓ Multi-provider AI: Anthropic / OpenAI / Gemini
✓ Dark / Light / System theme
✓ English / Turkish language
✓ Email notifications (SMTP)


SETTINGS
---------
Open the app → click Settings (⚙) in the sidebar:
- Enter your AI API key (Anthropic / OpenAI / Gemini)
- Set up email for reminders
- Choose theme and language
